# Описание структуры проекта

- `src/db/__main__.py` - точка входа
- `src/db/tui.py` - интерфейс в виде консоли, можно вводить команды и выводить информацию
- `src/db/backend/memory.py` - база данных, добавление и удаление записей, а также обновление и поиск
- `src/db/__init__.py` - файл для бекенда

## Функционал

Удаление и добавление записей, просмотр записей и поиск их по полям, обновление записей, проверка на ошибки.

## Как запустить

В корневой папке в терминале:

```bash
python -m src.db
```
