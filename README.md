# PIOA-123BV

Лабораторная работа по ООП и тестированию. Проект представляет собой простую in-memory систему для хранения книг с консольным интерфейсом.

## Структура проекта

```text
src/
  db/
    __main__.py
    tui.py
    backend/
      __init__.py
      errors.py
      memory.py
tests/
  __init__.py
  test_memory.py
README.md
```

## Что реализовано

- объектно-ориентированная реализация базы данных через классы `MemoryDatabase` и `BookTable`
- создание и выбор таблиц
- добавление, просмотр, поиск, обновление и удаление записей о книгах
- проверки на пустые поля, отрицательный год и дублирующийся `id`
- пользовательские исключения для типовых ошибок
- автоматические тесты для основной логики и части консольного интерфейса

## Формат записи

Каждая запись о книге хранится в виде кортежа:

```python
(book_id, title, author, year, genre)
```

## Запуск программы

Из корня проекта:

```bash
python -m src.db
```

## Запуск тестов

```bash
python -m unittest
```
